# Global Car Search

Buscador de autos en tiempo real: AutoScout24, Mobile.de, Coches.net, CarSales, OLX.

## Deploy

1. **Backend en Railway**: deploy de la carpeta `backend/`
2. **Frontend en GitHub Pages**: activar Pages en la carpeta `frontend/`

## API

```
GET /search?q=toyota+hilux&sources=autoscout24,mobilede,coches
```
